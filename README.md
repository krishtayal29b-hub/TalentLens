# TalentLens 🚀

**TalentLens** is an AI-powered recruitment and career platform designed to simplify the hiring process for both recruiters and job seekers. It combines intelligent resume analysis, AI assistance, and a modern job portal into a single application.

## 📌 Overview

TalentLens bridges the gap between recruiters and candidates by providing an efficient, transparent, and AI-driven hiring experience. Recruiters can easily publish job openings and manage applications, while candidates can build their profiles, apply for jobs, analyze their resumes, and receive career guidance through an AI-powered chatbot.

Whether you're searching for your dream job or looking for the perfect candidate, TalentLens makes recruitment faster, smarter, and more effective.

## ✨ Features

### 👨‍💼 For Recruiters

* Secure recruiter authentication
* Create and manage company profiles
* Post new job openings
* Edit or delete existing job listings
* View all applicants for each job
* Access candidate profiles and resumes
* Track application status
* Manage hiring workflow from a centralized dashboard

---

### 👩‍💻 For Candidates

* Secure user registration and login
* Create and update professional profiles
* Upload resumes
* Browse available job opportunities
* Apply for jobs with a single click
* Track application history and status
* Save and manage personal information

---

### 🤖 AI-Powered Resume Analysis

TalentLens includes an intelligent resume analyzer that evaluates resumes based on multiple professional criteria, such as:

* Resume Score
* ATS (Applicant Tracking System) Compatibility
* Skills Analysis
* Missing Keywords
* Formatting Suggestions
* Content Quality
* Professional Recommendations
* Resume Improvement Tips

This helps candidates optimize their resumes before applying for jobs.

---

### 💬 AI Career Assistant

An integrated AI chatbot assists users by providing:

* Resume improvement suggestions
* Career guidance
